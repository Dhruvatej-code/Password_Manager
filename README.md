# PassOP - Password Manager

PassOP is a password manager built using React, Tailwind CSS, Node.js, Express.js, and MongoDB.

This repository contains two versions of the project:

1. **Local Storage Version** – Stores passwords in the browser using Local Storage.
2. **MongoDB Version** – Stores passwords in a MongoDB database using a Node.js and Express.js backend.

## Features

* Save passwords
* View stored credentials
* Edit existing entries
* Delete passwords
* Responsive user interface

## Tech Stack

* React.js
* Tailwind CSS
* Node.js
* Express.js
* MongoDB
